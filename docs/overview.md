# Campfire 概要

## このアプリは何か

Campfire は、チーム内コミュニケーション向けの Web チャットアプリです。  
単一テナント構成で、1つのデプロイ先を1つの組織で利用する想定です。

## 主な機能

- 複数ルーム（アクセス制御あり）
- ダイレクトメッセージ
- ファイル添付とプレビュー
- メッセージ検索
- Web Push 通知
- `@mention`
- Bot 連携向け API

## 運用上の前提

- 初回起動時に管理者アカウントを作成します。
- 「公開」ルームは同一インスタンス内の全ユーザーから参照可能です。
- テナント分離が必要な場合は、インスタンスを分けてデプロイします。
